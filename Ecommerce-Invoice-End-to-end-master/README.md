## 📌 Project Overview

A full-stack data engineering + analytics project demonstrating real-world software practices (CI/CD, containerization, testing) alongside business intelligence. Covers the complete data lifecycle from collection to executive dashboards.

## 💡 Key Business Insights

| Metric | Finding |
|---|---|
| 📦 Top Products | High-volume SKUs identified for inventory optimization |
| 🌍 Top Markets | Regional revenue breakdown with growth trends |
| 🔄 Returns Analysis | Return rate patterns by category and season |
| 📅 Seasonality | Peak demand periods identified for supply planning |
| 💰 Revenue Trends | Month-over-month and YoY growth analysis |

## 📁 Project Structure

| Folder/File | Purpose |
|---|---|
| src/ | Core pipeline source code |
| notebooks/ | Exploratory analysis notebooks |
| tests/ | Unit and integration tests |
| docker/ | Container configuration |
| terraform/ | Infrastructure as code |
| web-scraping/ | Data collection scripts |
| ecomm_bi.pbix | Power BI dashboard |
| Makefile | Build automation commands |

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Data Collection | Python web scraping |
| Processing | Python, Pandas, NumPy |
| Infrastructure | Docker, Terraform |
| CI/CD | GitHub Actions, pre-commit |
| Visualization | Power BI Desktop |
| Testing | pytest |
