> **End-to-end ML pipeline: EDA to model training to Flask web app — predicting telecom churn with Power BI dashboard for business stakeholders.**

## 📌 Overview
Complete machine learning solution identifying at-risk telecom customers before they leave — deployed as a web app with a business-friendly Power BI dashboard.

## 💡 Key Churn Drivers
| Factor | Signal |
|---|---|
| Contract Type | Month-to-month customers churn 3x more |
| Monthly Charges | Over $70/mo increases churn by 40% |
| Tenure | First 12 months = highest risk window |
| Internet Service | Fiber optic users show highest churn |
| No Tech Support | Significantly higher churn without support |
| Payment Method | Electronic check = highest churn rate |

## 📁 Structure
| Folder/File | Description |
|---|---|
| app.py | Flask web application |
| notebook/ | EDA and model training notebooks |

| powerBI/ | Power BI dashboard |

Python, Pandas, Scikit-Learn (Logistic Regression, Random Forest, XGBoost), Flask, Power BI
