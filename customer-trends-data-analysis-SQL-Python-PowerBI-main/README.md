> **Three-tool pipeline: SQL segmentation + Python EDA + Power BI dashboard — uncovering shopping trends, seasonal patterns, and high-value customer segments.**

## 📌 Overview
Analyzes 3,900+ customer records across demographics, categories, seasons, and payment methods to drive targeted marketing and inventory decisions.

## 💡 Key Insights
| Insight | Finding |
|---|---|
| Top Segment | Ages 25-45 drive 55% of total revenue |
| Peak Season | Fall and Winter show 30% higher average spend |
| Payment | Credit card users have highest average order value |
| Subscription | Subscribers spend 2.5x more per visit |
| Top Category | Clothing dominates volume; Outerwear tops value |
| Reviews | 4+ star ratings = 40% higher repeat purchase rate |
