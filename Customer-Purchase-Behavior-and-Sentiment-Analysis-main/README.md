> **360-degree e-commerce analysis combining purchase behavior, RFM segmentation, and NLP sentiment scoring — with a live animated Power BI dashboard.**

## 📌 Overview
Fuses transactional data with customer review sentiment to deliver complete visibility into what customers buy, how they feel about it, and how to retain high-value segments.

## 💡 Key Insights
| Dimension | Finding |
|---|---|
| Top Segment | High-spend customers (top 20%) drive 60%+ revenue |
| Sentiment | Positive reviews correlate 3x with repeat purchase |
| Return Rate | Low-rated products have 2x higher returns |
| Category Leaders | Electronics & Clothing top volume; Jewelry tops value |
| RFM Segments | Champions and Loyal Customers identified for upsell |
