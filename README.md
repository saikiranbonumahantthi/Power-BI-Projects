### 2. 📉 Customer Churn Analysis
**Folder:** `Customer-Churn-Analysis-main`
- Identified churn patterns across customer segments
- Actionable insights for retention strategy
- **Tools:** Power BI, SQL, Python

**Folder:** `Customer-Purchase-Behavior-and-Sentiment-Analysis-main`

**Folder:** `Ecommerce-Invoice-End-to-end-master`

- **Tools:** Power BI, DAX, Power Query
